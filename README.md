### 🧩 What I Build

**🎨 Front-End**
Responsive, component-driven UIs — React, Next.js, and Vite-based apps with clean, functional design.

**⚙️ Back-End**
REST APIs and services in Python (Flask/FastAPI-style) and Node.js/Express, backed by SQL databases, with authentication and data-driven logic (including ML-model integration).

**☁️ Cloud Computing**
Containerized services with Docker, cloud deployment (AWS / Google Cloud Run), and cloud-hosted databases for scalable backend infrastructure.

**📡 IoT & Embedded Systems**
Register-level embedded C on AVR microcontrollers (ATmega328P — interrupts, timers, PWM) and ESP32-based IoT devices with MQTT connectivity for real-time, sensor-driven systems.

**☕ Also Explored**
Java & Spring Boot for backend services, and static site generation (Jekyll) for content-driven websites.
